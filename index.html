El Impostor

Juego pass-and-play para un solo dispositivo. Configura, revela roles con privacidad, juega contra el tiempo y vota al sospechoso.

Configuración

Mínimo 3 jugadores. Para tener impostores, necesitas al menos 4.

Resumen

Flujo: CONFIG → REVEAL → GAME/TIMER → VOTING → RESULTS
Privacidad: Mantén presionado para ver tu rol en cada turno.
Victoria: Civiles ganan si expulsan a todos los impostores. Impostores ganan si el tiempo se agota o igualan en número a los civiles.

Revelación de rol

Entrega el dispositivo al jugador indicado y mantén presionado para revelar su información.
